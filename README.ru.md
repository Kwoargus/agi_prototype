# 🧠 AGI Prototype: Constructional Approach

**Репозиторий находится на этапе активного проектирования. Мы ищем единомышленников для совместной разработки прототипа AGI на основе «конструкционного подхода».**

## Проблема
Современные архитектуры ИИ (трансформеры, MoE) великолепны в узких задачах, но не воспроизводят иерархическую организацию когнитивных механизмов, что, по нашему мнению, является необходимым условием для достижения общего интеллекта.

## Решение
Мы предлагаем **конструкционный подход**: создать нейросетевой аналог человеческого интеллекта, структурно повторяющий пять эволюционно обусловленных уровней психики: **рефлексы, инстинкты, эмоции, интеллект, сознание**. Каждый уровень отвечает за свой тип обработки сигналов, а ключевым механизмом является генерация новых знаний через поиск аналогий и модификацию моделей.

Подробности изложены в [статье](https://mtnip.elpub.ru/jour/article/view/35).

## Архитектура (кратко)
|-----------|---------------------------------------------------------------|------------------------------------------------------------|
| Уровень   | Функция | Техническая интерпретация                                                                                        |
|-----------|---------------------------------------------------------------|------------------------------------------------------------|
| Рефлексы  | Мгновенная реакция на пороговые стимулы                       | Быстрые детекторы аномалий / триггеры безопасности         |
| Инстинкты | Поведение на основе паттернов                                 | Библиотека обученных сценариев + прогноз последствий       |
| Эмоции    | Социальная синхронизация и оценка ценности                    | Модуль оценки «полезности» действий для группы             |
| Интеллект | Создание и использование динамических прогностических моделей | Движок генеративных моделей + механизм «интуиция+аналогия» |
| Сознание  | Мета-модель «Я-Группа-Мир»                                    | Контекстный менеджер, обеспечивающий целостность поведения |
|-----------|---------------------------------------------------------------|------------------------------------------------------------|

## Ключевые механизмы
- **Граф знаний**: хранение моделей и связей для поиска аналогий.
- **Динамические модели**: объекты и процессы кодируются как последовательности состояний.
- **Генерация новых знаний**: перебор, модификация и проверка существующих моделей для описания новых ситуаций.
- **Обучение**: комбинация генеративных сетей и генетических алгоритмов, где критерием отбора служит «способствует выживанию и развитию жизни» – это встраивает этическую направленность.
