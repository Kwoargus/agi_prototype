# 🧠 AGI Prototype: Constructional Approach

**The repository is in the active design phase. We are seeking like-minded collaborators to jointly develop an AGI prototype based on the "constructional approach."**

## Problem
Current AI architectures (Transformers, MoE) excel at specific tasks, but they do not reproduce the hierarchical organization of cognitive mechanisms, which we believe is a prerequisite for achieving general intelligence.

## Solution
We propose a **constructional approach**: create a neural network analogue of human intelligence, structurally replicating the five evolutionarily determined levels of the psyche: **reflexes, instincts, emotions, intellect, and consciousness**. Each level is responsible for its own type of signal processing, and the key mechanism is the generation of new knowledge through analogy search and model modification.

Details are provided in the [article](https://mtnip.elpub.ru/jour/article/view/35).

## Architecture (briefly)
|---------------|--------------------------------------------------|----------------------------------------------------------------|
| Level         | Function                                         | Technical Interpretation                                       |
|---------------|--------------------------------------------------|----------------------------------------------------------------|
| Reflexes      | Instant response to threshold stimuli            | Fast anomaly detectors / safety triggers                       |
| Instincts     | Pattern-based behavior                           | Library of learned scenarios + consequence prediction          |
| Emotions      | Social synchronization and value assessment      | Module for assessing the "usefulness" of actions for the group |
| Intelligence  | Creation and use of dynamic predictive models    | Generative model engine + "intuition + analogy" mechanism      |
| Consciousness | "Self-Group-World" meta-model                    | Context manager ensuring behavioral integrity                  |
|---------------|--------------------------------------------------|----------------------------------------------------------------|

## Key Mechanisms
- **Knowledge Graph**: storing models and connections for finding analogies.
- **Dynamic models**: objects and processes are encoded as sequences of states.
- **Generation of new knowledge**: iterating, modifying, and validating existing models to describe new situations.
- **Learning**: a combination of generative networks and genetic algorithms, where the selection criterion is "promotes the survival and development of life" – this embeds an ethical focus.
